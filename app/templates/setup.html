What is this for?

This project is open-source and that means you are able to host a copy of this website on your own web server.

The GitHub repository for this project can be found over here, a more technical set-up guide can also be found there.

Pre-requisites:

You will need several things already installed on your machine before you can continue

  1. python3
  2. All the modules required:
    pip install flask flask-sqlalchemy flask-migrate sqlalchemy gunicorn numpy matplotlib pandas psycopg2 python-dotenv pygal
  3. Heroku
  4. PostgreSQL

Installation Guide:

Quick installation on a local machine:

  1. Clone the repository
    git clone https://github.com/woojiahao/pds-analysis.git
  2. Navigate into the project folder
    cd pds-analysis
  3. Start the flask application which can be found at http://127.0.0.1:5000/
    flask run

Hosting on a local machine:

  1. Clone the repository
    git clone https://github.com/woojiahao/pds-analysis.git
  2. Navigate into the project folder
    cd pds-analysis
  3. Create a database in PostgreSQL called 'pds' using pgAdmin
  4. Load the data into the database
    flask shell
    >>> db_manager.load_data()
  5. Load the plots into the project
    >>> plot_loader.load_plots()
  6. Exit the flask shell
    >>> exit()
  7. Run the application
    flask run

Hosting on Heroku:

  1. Clone the repository
    git clone https://github.com/woojiahao/pds-analysis.git
  2. Navigate into the project folder
    cd pds-analysis
  3. Create a local database in PostgreSQL called 'pds' using pgAdmin
  4. Load the data into the local database
    flask shell
    >>> db_manager.load_data()
  5. Load the plots into the project
    >>> plot_loader.load_plots()
  6. Exit flask shell
    >>> exit()
  7. Create a Heroku application, you can optionally give it a custom name
    heroku create <optional custom name>
  8. Add the Heroku-Postgres add-on to the application
    heroku addons:create heroku-postgresql:hobby-dev
  9. Push the repository to Heroku
    git add .
    git commit -m "Setting up"
    git push -u heroku master
  10. Load the database into Heroku
    heroku run flask shell
    >>> db_manager.load_data()
  11. Exit the shell
    >>> exit()
  12. Activate the web dyno
    heroku ps:scale web=1
  13. Open the website
    heroku open
