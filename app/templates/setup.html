What is this for?

This project is open-source and that means you are able to host a copy of this website on your own web server.

The GitHub repository for this project can be found over here, a more technical set-up guide can also be found there.

Pre-requisites:

You will need several things already installed on your machine before you can continue

  1. python3
  2. All the modules required:
    pip install flask flask-sqlalchemy flask-migrate sqlalchemy gunicorn numpy matplotlib pandas psycopg2 python-dotenv pygal
  3. Heroku
  4. PostgreSQL

Installation Guide:

Hosting on a local machine:

  1. Clone the repository
    git clone https://github.com/woojiahao/pds-analysis.git
  2. Navigate into the project folder
    cd pds-analysis
  3. Create a database in PostgreSQL called 'pds' using pgAdmin
  4. Load the data into the database
    flask shell
    >>> Manager().load_data()
  5. Exit the flask shell
    >>> exit()
  6. Run the application
    flask run

Hosting on Heroku:

  1. Clone the repository
    git clone https://github.com/woojiahao/pds-analysis.git
  2. Navigate into the project folder
    cd pds-analysis
  3. Create a Heroku application, you can optionally give it a custom name
    heroku create <optional custom name>
  4. Add the Heroku-Postgres add-on to the application
    heroku addons:create heroku-postgresql:hobby-dev
  5. Push the repository to Heroku
    git push -u heroku master
  6. Load the database
    heroku run flask shell
    >>> Manager().load_data()
  7. Exit the shell
    >>> exit()
  8. Activate the web dyno
    heroku ps:scale web=1
  9. Open the website
    heroku open
